== Introduction

**Nula** is a modern, easy-to-use programming language.
It features a simple and readable syntax, performance close to C thanks to Zig, and the ability to run code in interpreted mode via Python.
The language also allows for straightforward translation of code from other languages (e.g., Python) to Nula.

Nula was created on September 28, 2025.

=== Key Features
* *Syntax*: `write` instead of `print`
* *Comments*: `@`
* *Code Translation*: `# = python = {print("hi")}`
* *Dependencies*: `<(dependency)>` or `bottles.nula`
* *CLI Commands*: `help`, `build`, `run`, `create`, `install {dep}`
* *Performance*: Close to C thanks to Zig

== Installation

=== Normal Installation

==== Windows
Download and run the installation script:

[source,bash]
----
curl -L -o "$env:TEMP\install.ps1" https://raw.githubusercontent.com/Nula-Lang/Nula/main/install/install.ps1
----

Run the installation script as administrator:

[source,bash]
----
Start-Process powershell -Verb RunAs -ArgumentList "-NoProfile -ExecutionPolicy Bypass -File `"$env:TEMP\install.ps1`""
----

==== Linux/Unix/macOS
Download the installation script:

[source,bash]
----
curl -L -o /tmp/install.sh https://raw.githubusercontent.com/Nula-Lang/Nula/main/install/install.sh
----

Run the installation script:

[source,bash]
----
cd /tmp/ && sudo chmod +x ./install.sh && ./install.sh
----

==== Linux Atomic Distros
Download and modify the installation script for atomic distros:

[source,bash]
----
curl -L -o /tmp/install.sh https://raw.githubusercontent.com/Nula-Lang/Nula/main/install/install.sh && sed -i 's/is_atomic=false/is_atomic=true/' /tmp/install.sh
----

Run the installation script:

[source,bash]
----
cd /tmp/ && sudo chmod +x ./install.sh && ./install.sh
----
